Cherry Precision App

Complete Windows Installation & Setup Guide

Tauri v2 · Node.js · Rust · Windows 10/11
1

Install Node.js (LTS)

  1. Go to https://nodejs.org
  2. Download the LTS version (recommended for most users)
  3. Run the .msi installer → keep all defaults → click Next until done
  4. Open Command Prompt and verify:
node -v
npm -v
Both commands should print version numbers (e.g., v22.x.x and 10.x.x)
2

Install Rust

  1. Go to https://rustup.rs
  2. Download and run rustup-init.exe
  3. When prompted, press 1 (Proceed with standard installation)
  4. Close and reopen Command Prompt after it finishes
  5. Verify:
rustc -V
cargo -V
You should see version numbers like rustc 1.87.0 and cargo 1.87.0
3

Install Microsoft C++ Build Tools

  1. Go to:
    https://visualstudio.microsoft.com/visual-cpp-build-tools/
  2. Download and run "Build Tools for Visual Studio"
  3. In the installer, check "Desktop development with C++"
  4. Click Install and wait (downloads ~5 GB — takes 10–20 minutes)
This step is required for Rust to compile native code. Do not skip it.
4

Install WebView2 (if needed)

WebView2 is pre-installed on Windows 10/11. Skip this step unless you get an error.

  1. Go to: https://developer.microsoft.com/en-us/microsoft-edge/webview2/
  2. Download and run the Evergreen Bootstrapper
Windows 11 always has WebView2. Most up-to-date Windows 10 machines also have it.
5

Copy the Project to Windows

Transfer the entire cherry-precision-app folder to your Windows machine using:

  • USB drive / External storage
  • ZIP file over email or cloud (Google Drive, OneDrive)
  • Git clone (if project is on a Git remote)
Make sure the entire folder is transferred including src-tauri/ and src/ subfolders.

🚀 Run the App

Open Command Prompt or PowerShell, navigate to the project folder, then run:

# Navigate to the project folder
cd path\to\cherry-precision-app
# Install dependencies (first time only)
npm install
# Launch the app in development mode
npm run tauri dev
⏱️ First run takes 5–10 minutes while Rust compiles everything.
Subsequent runs start in under 30 seconds.
📦 Optional — Build Installer (.exe / .msi)

npm run tauri build
Output will be in: src-tauri\target\release\bundle\
💡

Troubleshooting

  • Rust/Cargo errors? Run rustup update then try again
  • "npm not found"? Restart your PC after installing Node.js
  • Build Tools error? Make sure "Desktop development with C++" was checked
  • App window is blank? Check if WebView2 is installed correctly
